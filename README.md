### What I learned

While building this project, I practiced creating a centered card layout with Flexbox and styling a simple responsive component with clean spacing and typography.

One important thing I learned was how asset paths behave differently in local development versus GitHub Pages deployment. Using an absolute path like `/images/image-qr-code.png` can break on project-based GitHub Pages sites, while a relative path works correctly.

```html
<img class="qrcode" src="./images/image-qr-code.png" alt="QR code" />
```
